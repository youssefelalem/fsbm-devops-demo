🔄 Processus

Pipeline Data Science

De la donnée brute à la prédiction : chaque étape de notre méthodologie.

📥

Collecte des Données

Dataset de 10 000 étudiants marocains avec 286 variables couvrant les aspects socio-économiques, académiques et comportementaux.

Morocco_Student_Data_Pool.csv
🧹

Nettoyage & Préparation

Correction du décalage de colonnes, suppression de 18 colonnes vides, imputation intelligente (médiane/mode), encodage des variables catégorielles.

clean_data.ipynb
📊

Analyse Exploratoire (EDA)

Distributions, corrélations, détection d'outliers, analyse bivariée et multivariée. Identification des features les plus prédictives.

Advanced_EDA_Morocco_Students.ipynb
🤖

Modélisation ML

Comparaison de 8 algorithmes : Régression Linéaire, Ridge, Lasso, ElasticNet, Random Forest, Gradient Boosting, XGBoost et MLP. Optimisation avec RandomizedSearchCV.

ML_student_performance_VF.ipynb
📈

Évaluation & SHAP

Métriques : RMSE, MAE, R². Analyse des résidus, courbes d'apprentissage, et interprétabilité avec SHAP Values.

evaluation.ipynb
🚀

Déploiement API

Le meilleur modèle (Gradient Boosting Tuné) est sauvegardé et servi via une API Flask pour des prédictions en temps réel.

api.py + Flask
